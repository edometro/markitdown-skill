# MarkItDown スキル (README.md)

このスキルは、PDF、Officeドキュメント（DOCX, PPTX, XLSX）、HTML、画像、音声などの多様なメディアファイルを、ローカルの `markitdown` 変換環境を使用してMarkdown形式に高精度で変換・整形するためのスキルです。

## 主な機能と特徴
- **マルチフォーマット対応**: PDF、Word、PowerPoint、Excel、HTML、画像、オーディオ等のシームレスなMarkdown変換。
- **ローカル実行による安全性**: 外部のAPIに依存せず、ローカルに構築された環境上でセキュアに処理を行います。
- **AIによる高度なレイアウト最適化**: 単なるテキスト抽出にとどまらず、スライド構造の見出し化、箇条書きの標準化、表や図のMarkdown表現など、人間が読みやすい形式へのインテリジェントな後処理（リファインメント）を行います。

## 基本的な仕組み
本スキルは、スキルディレクトリ内に配置されたラッパースクリプト `bin/markitdown.sh` を通じて、ローカルの `markitdown` 環境を呼び出して動作します。

### 環境のカスタマイズ性（汎用的な設計）
デフォルトでは `~/markitdown` にチェックアウトされたリポジトリと、その配下の仮想環境（`.venv`）を利用しますが、環境変数 `MARKITDOWN_REPO` や `MARKITDOWN_VENV` を設定することで、任意のパスへ柔軟にカスタマイズすることが可能です。

## 初期設定と自動環境構築 (Zero-Setup & Prerequisites)

本スキルを利用するにあたり、**人間（ユーザー）による手動の初期設定や環境構築コマンドの実行は原則として一切不要です**。

### AIエージェントによる自動セットアップ
スキル実行時、または環境破損時にローカル環境が未構築であることを検知した場合、**AIエージェント自身が自律的に以下の環境構築および修復をバックグラウンドで自動実行します**。
- `markitdown` リポジトリの配置およびGitHubからの自動クローン
- リポジトリ直下への `venv` による専用のPython仮想環境の自動作成
- 必要なライブラリおよび `packages/markitdown[all]` の自動インストール

### システムに求められる推奨構成（オプション）
PDFのメタデータ解析やマルチメディアの高度なテキスト抽出機能を利用するため、ホストシステムに以下のツールがインストールされていることが推奨されます。AIエージェントが必要性を検知した場合、推奨メッセージを提示します。
- **exiftool**: PDFや画像等の詳細なメタデータ情報の抽出に使用
- **ffmpeg**: オーディオ/ビデオ等の解析・テキスト変換に使用
